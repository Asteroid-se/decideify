# 🚀 Decidefy: The Ultimate Decision Hub 🎡

**Decidefy**, kararsızlık anlarını eğlenceye dönüştüren, içinde interaktif bir karar çarkı ve refleks tabanlı mini oyunlar barındıran modern bir web uygulamasıdır.

**website:** https://asteroid-se.github.io/decideify/

## ✨ Özellikler

### 🎡 Decidefy Çarkı
*   **Dinamik Seçenek Yönetimi:** Kendi seçeneklerini ekle, çıkar ve anında çarkta gör.
*   **Gerçekçi Fizik Motoru:** CSS `cubic-bezier` ile yavaşlayan, doğal dönüş animasyonu.
*   **Ses Deneyimi:** Her dilim geçişinde "Tık" sesi ve zafer anında "Tada" efekti.
*   **Görsel Şölen:** Arka planda süzülen balonlar ve kazanan açıklandığında patlayan konfetiler.

### 🕹️ Arcade Mode: Neon Dash
*   **Refleks Testi:** 10 saniye içinde en çok tıklamayı yapmaya çalış!
*   **High Score Sistemi:** En yüksek skorun `localStorage` ile tarayıcında saklanır.
*   **Neon Efektleri:** Her tıklamada değişen parlak neon ışıkları ve görsel geri bildirim.

## 🛠️ Teknik Detaylar

*   **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla).
*   **Grafikler:** `Canvas API` kullanılarak dinamik olarak çizilen çark dilimleri.
*   **Ses:** `Web Audio API` (Oscillator) ile sıfır dış dosya bağımlılığıyla üretilen sesler.
*   **Animasyon:** Tailwind JIT ve Özel CSS Keyframes.

## 🚀 Kurulum

Bu projeyi yerelinde çalıştırmak için herhangi bir sunucuya veya kuruluma ihtiyacın yok:

1.  Repoyu klonla:
    ```bash
    git clone https://github.com/kullaniciadi/decidefy.git
    ```
2.  `index.html` dosyasını tarayıcında aç.
3.  Kararını ver veya Arcade modunda rekor kır!
